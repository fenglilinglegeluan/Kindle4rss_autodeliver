# Kindle4rss_autodeliver

基于 GitHub Actions 的 kindle4rss 免费推送脚本。

## 使用方法

1. `Fork` 仓库
2. 设置仓库的 Actions Secrets
   添加用户名 `USERNAME` 和密码 `PASSWORD`
   | Name | Value |
   | :---: | :---: |
   | USERNAME | Kindle4rss 用户名 |
   | PASSWORD | Kindle4rss 密码 |
3. 开启 GitHub Actions
4. 每天 11:00 <sup>3:00 UTC</sup> 定时自动运行
